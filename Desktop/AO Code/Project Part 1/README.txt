To run this code in Terminal or the windows equivalent to windows you enter:

python (The file path to test_decision.py)

After this it will prompt you to enter the training set and the test set to do this you enter
the file path to the trainer file and the test set file with a space between the files like this:

(The file path to trainer.txt) (The file path to testSet.txt)

the code will then run and it should output 1

you can change the test set to have different orders of 1 and 0s and it will output different results.
